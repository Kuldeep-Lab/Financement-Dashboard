# React + Vite

For the setup clone this repo in your code editor and then run this command to install all the necessary dependency 

npm install 

after installation complete run this command to start your local server

npm run dev 

# App features 

1. Finance dashboard 
2. Collapsible Sidebar allow user to switch between different pages.
3. Stats cards displaying total balance, income, expenses and savings. 
4. donut chart displaying category wise expenese.
5. Pie chart displaying yearly based balance trends.
6. Insights section displaying top category wise expenese, number of transactions and savings.
7. Transactions table displaying transactions details such as date , category, amount and type.

# Main features 

1. At the top of finance dashboard in navbar we have drop-down option that allow user to switch roles between viewer and admin 

Note - Only admin has the authority to make changes in the transaction details or add new transaction details. Viewer can only read.

2. based on transaction data both stats cards and donut chart displaying the details if you edit or add new transaction you can immediately can see the effects in the stats , donut chart and in the Insights section.

3. Responsive UI accross all devices laptop, Tablet and mobile user friendly interface easy to navigate.

Note - In this project i only create the dashboard page not the rest of the pages so you can switch between different pages but there's no UI in other pages i just added the rest of the pages for the navigation.
